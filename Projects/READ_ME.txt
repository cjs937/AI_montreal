Github link: https://github.com/cjs937/AI_montreal/tree/master/Projects/pathfinding

In order to compile the project DeanLib and the directory containing the allegro5 folder need to be in the directory above the Projects directory. After that everything should work fine.

Inputs:
LMB - Sets goal point for the path
A - Switches algorithm to A*
D - Switches algorithm to Dijkstra
Esc - Closes and exits the game
